  # Getting started

  ## Run Storybook

  ```bash
  pnpm storybook
  ```

  ## Add a component

  1. Create the component under `src/ui/<category>/`.
  2. Add a `*.stories.tsx` file with a title like `Components/Button`.
  3. Import global styles via `src/storybook-setup.ts` (already wired in `.storybook/preview.tsx`).

  ## Theming

  - Design tokens live in `src/index.css` as `--sui-*` variables.
  - Use Tailwind utilities (`bg-primary`, `text-foreground`, etc.) in most components.
  - Wrap MUI stories with `MuiThemeProvider` (see Dialog / Select stories).

  ## Tests

  ```bash
  pnpm test:storybook
  ```
